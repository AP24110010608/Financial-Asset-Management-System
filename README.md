# Finance and Asset Management System (FAMS)

## Project Overview
Finance and Asset Management System (FAMS) is a DBMS project designed to manage the lifecycle of financial and physical assets in an organization.

The system handles:
- Asset procurement
- Portfolio management
- Financial accounts
- Transactions
- Depreciation tracking
- Disposal and scrap management

## Technologies Used
- MySQL / Oracle SQL
- draw.io (ER Diagram)

## Main Entities
- ASSET
- PORTFOLIO
- SUB_PORTFOLIO
- PURCHASE_ORDER
- LOCATION
- FINANCIAL_ACCOUNT
- TRANSACTION
- DEPRECIATION
- CLIENT
- VENDORS
- DISPOSAL
- SCRAP_YARD

## Features
- Normalized database up to 3NF
- SQL Queries using JOINs, Aggregate Functions, and Subqueries
- Database Views for reporting
- Asset lifecycle tracking
